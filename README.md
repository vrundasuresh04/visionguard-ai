# 🛡 VisionGuard AI – Real-Time Face Mask Detection System

## 🚀 Overview
VisionGuard AI is a real-time AI-powered web application that detects whether a person is wearing a mask or not using computer vision and deep learning.

It provides live camera monitoring, detection alerts, and an interactive dashboard for analytics.

---

## 🔥 Features
- 🎥 Real-time webcam detection
- 🤖 YOLOv8-based object detection
- 📊 Live dashboard with analytics
- 🔊 Sound alert for no-mask detection
- 📁 CSV report generation
- 🌐 Flask-based web interface
- 🔐 User authentication system

---

## 🧠 Tech Stack
- Python
- Flask
- YOLOv8 (Ultralytics)
- OpenCV
- SQLite
- Chart.js
- HTML, CSS, JavaScript
